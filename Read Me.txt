Install Localhost:

1. Buat database dengan nama nataibaru.
2. Extract File nataibaru.rar yang didownload kedalam directori:
   xampp\htdocs (jika menggunakan XAMPP)
3. Buka http://localhost/phpmyadmin. Lalu import nataibaru.sql kedalam database yang ada buat.
4. Pada Browser Buka URL http://localhost/nataibaru/
5. Selesai.
